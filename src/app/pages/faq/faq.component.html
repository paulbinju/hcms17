FAQ


How to create an Article for Tradearabia and Magazines?

On the left hand side menu click on Latest > Add new or Latest > View all page right side to you have a button to create new article.

If you are creating for daily news select Tradearabia from the publication and select corresponding magazine from subsection.

If you are creating for monthly / issue article, then select corresponding Publication and select issue.


How to add / edit dropdown values?

On the left hand side menu click on Settings select the section and add / edit values, it will be then reflected in the dropdown list.


How to add / edit Users?

Only Administrator has the permission to add or edit user / password. Contact Administrator to add or edit user.
